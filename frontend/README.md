# 📌 Project Description

This is a Smart YouTube Clone developed as a Final Year + Internship Project.  
The platform includes AI-based theming, smart authentication, intelligent comment moderation, and modern video platform features.

---

# ✨ Key Features

## 🎨 Smart Theme System
- Time-based theme switching
- Location-based theme
- Dark & Light Mode

Condition:
- 10:00 AM — 12:00 PM IST  
- South India → Light Theme  
- Other time/location → Dark Theme

---

## 🔐 Smart Authentication

Region-based authentication:

South India:
- Email OTP verification

Other States:
- Mobile OTP verification

Supported States:
- Tamil Nadu
- Kerala
- Karnataka
- Andhra Pradesh
- Telangana

---

## 💬 Smart Comment System

- Comment Translator
- Like / Dislike Comments
- Auto Delete after 2 dislikes
- City Detection
- Special Character Block
